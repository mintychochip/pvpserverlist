# GuildPost Brand Guidelines

## Brand Overview

**GuildPost** is a modern game server discovery platform. We connect players with their next multiplayer home across Minecraft, Rust, CS2, and future games.

**Tagline:** *Find your next multiplayer home*

---

## Logo

### Primary Logo
- **Dark version:** `../logo/logo-dark.svg` (use on light backgrounds)
- **Light version:** `../logo/logo-light.svg` (use on dark backgrounds)

### Icon Only
- **SVG:** `../icons/logo-icon.svg`
- **PNG sizes:** 180px, 192px, 512px

### Usage Rules
- Maintain clear space around the logo (minimum height of the "G")
- Do not stretch, rotate, or alter the logo colors
- Use the dark logo on light backgrounds, light logo on dark backgrounds

## Typography

### Wordmark
- **Primary:** Elegant serif (Freight Display, Tiempos Headline, or Garamond)
- **Alternative:** Clean sans-serif for "guildpost" lowercase usage

### UI Text
- **Headings:** System sans-serif stack

## Brand Voice

- **Clean & Modern:** No clutter, straightforward communication
- **Community-First:** Focus on players and server communities
- **Game Agnostic:** Minecraft is our starting point, but we serve all multiplayer games
- **Professional but Approachable:** Trustworthy without being stuffy

### Messaging Do's
- "Discover game servers"
- "Find your next multiplayer home"
- "Browse thousands of servers"
- "Server discovery platform"

### Messaging Don'ts
- "PvP server list" (too narrow)
- "Minecraft server list" (we're more than a list)
- "Best PvP servers" (excluding other playstyles)
