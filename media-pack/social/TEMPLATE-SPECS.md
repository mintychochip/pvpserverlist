# GuildPost Social Media Templates

## Template Specifications

### 1. Twitter/X Post Templates

#### Square Post (1080×1080)
- **Background:** Deep navy `#0a0f1c`
- **Logo placement:** Top center, 120px width
- **Content area:** Center, 60% of canvas
- **Hashtag area:** Bottom, 80px height
- **Safe zone:** 100px padding all sides

#### Portrait Post (1080×1350)
- **Background:** Deep navy `#0a0f1c` with subtle gradient
- **Logo placement:** Top left, 100px width
- **Headline area:** Upper third, 44px font
- **Content area:** Middle 50%
- **CTA area:** Lower third with teal accent `#00f5d4`

#### Story/Reels (1080×1920)
- **Background:** Full gradient or image
- **Logo placement:** Top center, small watermark
- **Content:** Vertical stack, centered
- **Swipe indicator:** Bottom with animation cue

### 2. LinkedIn Templates

#### Company Banner (1584×396)
- **Background:** Dark navy `#0a0f1c`
- **Logo:** Left side, 200px height
- **Wordmark:** Centered right
- **Tagline:** "Find your next multiplayer home"

#### Post Template (1200×627)
- **Background:** Solid or subtle texture
- **Header:** Icon + "GuildPost" left aligned
- **Body:** 2-3 lines of content
- **Footer:** Link preview style

### 3. Instagram Templates

#### Feed Post (1080×1080)
- Same as Twitter/X square
- Story highlight style with border

#### Story Template (1080×1920)
- Full screen design
- Top: Logo watermark
- Middle: Dynamic content area
- Bottom: "Swipe up" or link sticker area

#### Reels Cover (1080×1920)
- Title card style
- Big bold typography
- Duration indicator

### 4. Discord Assets

#### Server Banner (960×540)
- Use existing `../banners/banner-discord.png`
- Alternative: Animated GIF version

#### Invite Splash (1920×1080)
- Full brand treatment
- Server stats overlay ready
- Join button integration

#### Emoji Pack
- Logo mark simplified (128×128)
- Network node icon
- Game mode icons (Minecraft block, Rust gear, CS2 target)

### 5. YouTube Templates

#### Thumbnail (1280×720)
- **Background:** Dark navy with subtle pattern
- **Logo:** Top left corner, 80px
- **Title:** Bottom third, bold white text with shadow
- **Accent:** Teal `#00f5d4` underline or highlight
- **Avatar:** Bottom right, circular mask

#### Channel Banner (2560×1440)
- **Safe zone:** 1546×423 (centered)
- **Logo:** Left side of safe zone
- **Tagline:** Right side of safe zone
- **Social links:** Bottom right outside safe zone

#### End Screen (1920×1080)
- **Subscribe button area:** Centered bottom
- **Video thumbnail areas:** Left and right (25% each)
- **Logo:** Top center, subtle

## Animation Guidelines

### Logo Animation
- **Duration:** 1.5 seconds
- **Easing:** ease-out
- **Effect:** Network nodes connect sequentially
- **Format:** Lottie JSON or CSS animation

### Transition Effects
- **Fade:** 300ms ease-in-out
- **Slide:** 400ms cubic-bezier(0.4, 0, 0.2, 1)
- **Scale:** 200ms ease-out

---

## Export Settings

### Static Images
- **Format:** PNG for graphics, JPG for photos
- **Resolution:** 72 DPI (screen), 300 DPI (print)
- **Color space:** sRGB

### Animated Assets
- **GIF:** 128 colors, 15fps max for UI
- **Lottie:** JSON with embedded images
- **Video:** H.264, 1080p, 30fps
