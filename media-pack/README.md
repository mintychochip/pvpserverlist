# GuildPost Media Pack

Complete brand asset package for GuildPost — the game server discovery platform.

## 📦 What's Included

```
media-pack/
├── logo/                    # Full wordmarks and variants
│   ├── logo-dark.svg       # Primary dark logo (transparent)
│   ├── logo-light.svg      # Light variant for dark backgrounds
│   ├── logo-dark.png       # Raster fallback
│   └── ...
├── icons/                   # Favicons and app icons
│   ├── favicon.svg         # Scalable favicon
│   ├── logo-icon-192.png   # Android/PWA icon
│   ├── logo-icon-512.png   # PWA splash
│   └── ...
├── banners/                 # Social and web banners
│   ├── og-image-new.png    # OpenGraph / social preview
│   ├── banner-discord.png  # Discord server banner
│   └── ...
├── social/                  # Social media templates
│   └── TEMPLATE-SPECS.md   # Specifications for all platforms
└── brand-guidelines/        # Brand documentation
    ├── README.md           # Quick brand guide
    └── ASSET-INVENTORY.md  # Complete asset reference
```

## 🚀 Quick Start

### Website Implementation
```html
<!-- Favicon -->
<link rel="icon" type="image/svg+xml" href="/favicon.svg">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">

<!-- OpenGraph -->
<meta property="og:image" content="https://guildpost.tech/og-image-new.png">

<!-- App Icons -->
<link rel="apple-touch-icon" sizes="180x180" href="/logo-icon-180.png">
<link rel="icon" sizes="192x192" href="/logo-icon-192.png">
```

### Social Media
- **Twitter/X Header:** Use `banners/header-twitter.png`
- **Discord Banner:** Use `banners/banner-discord.png`
- **Link Previews:** Use `banners/og-image-new.png`

## 📋 Brand Voice

- **Clean & Modern:** Straightforward, no clutter
- **Community-First:** Players and server communities
- **Game Agnostic:** Minecraft today, more games tomorrow
- **Professional but Approachable**

**Messaging:**
- ✓ "Discover game servers"
- ✓ "Find your next multiplayer home"
- ✓ "Server discovery platform"
- ✗ "PvP server list" (too narrow)
- ✗ "Minecraft server list" (we're more than a list)
