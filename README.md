# guildpost-astro

GuildPost - Find Your Server. A modern server browser for game servers.

## Brand Colors

- Background: `#0a0a0f`
- Primary (Cyan): `#00f5d4`
- Secondary (Pink): `#ff3864`
- White: `#ffffff`
- Muted Text: `#a0a0b0`

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The site is built for static export to GitHub Pages.
